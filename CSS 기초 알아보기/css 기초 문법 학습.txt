css 작성법은 아래와 같다.

선택자 {
    선언 블록
}

main {
    panding : 10px
}
얼핏 이런식이다.

선택자 종류는 다음과 같다.
1. 요소 선택자 : 요소명 { 선언 블록 }
2. 아이디 선택자 : #아이디명 { 선언 블록 }
3. 클래스 선택자 : .클래스명 { 선언 블록 }

선언블록에는 스타일 속성을 나열한다. 각 속성은 '이름', '쌍점 : ', '값'으로 구성된다.

css우선순위는 다음과 같다.

1. 중요성 -> 속성값 뒤에 !important 를 붙인 속성, 모든 스타일 보다 우선시한다.
2. 인라인 -> html 요소의 style 속성을 직접 지정한 스타일
3. 미디어 타입 -> 특정 css가 정의되어 있지 않다면 속성 정의는 모든 미디어 타입에 적용된다.
4. 사용자 정의 -> 사용자 정의 css가 html문서에 있는 css보다 우선한다.
5. 선택자 특정성 -> 특정 컨텍스트 선택자 (#heading p)는 일반 정의보다 우선한다.
6. 규칙 순서 -> 마지막에 선언한 규칙이 우선순위가 가장 높다.
7. 부모 상속 -> 속성을 지정하지 않으면 부모 요소를 상속한다.
8. html 문서 안에서의 css 속성 -> css 규칙이나 css 인라인 스타일은 기본 브라우저 값을 덮어쓴다.
9. 브라우저 기본값 -> 가장 낮은 우선순위, 브라우저 기본값은 w3c 초기값 사양에 정의되어 있다.
